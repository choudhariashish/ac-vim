# ac-vim
## VIM plugin configuration.

## Installation.
- sudo apt-get install ripgrep
- sudo apt-get install fzf
- Clone the repository
- cp ac-vim/.vimrc ~/
- cp -r ac-vim/.vim ~/
- Open=> vim ~/.vimrc and enter command :PlugInstall
- Open any file=> main.cpp and try Ctrl-r and Ctrl-f keys
- Happy navigating !
